## Program12 : Write a program to use switch statement
```c
#include <stdio.h>
int main()
{
int day = 4;
switch (day) 
{
case 6:
printf("Today is Saturday");
break;
case 7:
printf("Today is Sunday");
break;
default:
printf("Looking forward to the Weekend");
}
return 0;
}
```
**OUTPUT** :
```
Looking forward to the Weekend
```
